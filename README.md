# Cordova Anatomical Plugin

A plugin that presents a 3D representation of an anatomy view after being passed the appropriate JSON string.

## Using

Create a new Cordova Project

    $ cordova create anatomicalplugin com.example.anatomicalplugin AnatomicalPluginExample

Install the plugin

    $ cd anatomicalplugin
    $ cordova plugin add https://github.com/GobaLLC/pp-anatomical-plugin.git
